เซต (Set)

เซต คือกลุ่มของวัตถุหรือสมาชิกที่ชัดเจนและไม่ซ้ำกัน เราเขียนเซตโดยใช้วงเล็บปีกกา {} และสมาชิกแต่ละตัวเรียกว่า "สมาชิกของเซต" เซตมีหลายประเภท เช่น เซตจำกัด (มีสมาชิกจำนวนจำกัด) และเซตอนันต์ (สมาชิกไม่จำกัด) การดำเนินการบนเซตที่สำคัญได้แก่ ยูเนียน (∪) คือรวมสมาชิกทั้งหมด อินเตอร์เซคชัน (∩) คือสมาชิกที่อยู่ร่วมกัน และเซตย่อย (⊆) คือเซตที่สมาชิกทั้งหมดอยู่ในเซตอื่น

ตัวอย่าง: ถ้า A = {1, 2, 3} และ B = {2, 3, 4}
ยูเนียน A ∪ B = {1, 2, 3, 4}
อินเตอร์เซคชัน A ∩ B = {2, 3}
โจทย์ 1: ให้เซต A = {1, 3, 5, 7} และเซต B = {3, 4, 5, 6} จงหา A ∩ B คือเซตใด?
เฉลย: A ∩ B = {3, 5} เพราะเป็นสมาชิกที่อยู่ในทั้ง A และ B
โจทย์ 2: เซตใดต่อไปนี้คือเซตว่าง (ไม่มีสมาชิก)?
เฉลย: เซตว่างคือเซตที่ไม่มีสมาชิก เขียนแทนด้วย { } หรือ ∅

ตรรกศาสตร์ (Logic)

ตรรกศาสตร์เป็นสาขาคณิตศาสตร์ที่ศึกษาเกี่ยวกับความถูกต้องของเหตุผลและรูปแบบการสรุปความรู้ต่างๆ โดยใช้สัญลักษณ์แทน เช่น “และ” (∧), “หรือ” (∨), “ไม่” (¬) ข้อความประเภท “ถ้า...แล้ว...” เรียกว่า เงื่อนไขแบบมีเหตุมีผล ตรรกศาสตร์ช่วยให้เราวิเคราะห์และแก้ปัญหาอย่างเป็นระบบและมีเหตุผล

ตัวอย่าง: ประโยค “ถ้า ฝนตก ฉันจะไม่ไปโรงเรียน” แปลว่า “ถ้าฝนตก (p) แล้ว ฉันไม่ไปโรงเรียน (q)” ซึ่งเขียนในตรรกศาสตร์ได้ว่า p → q
โจทย์ 3: ข้อความ “ถ้า ฝนตก ฉันจะไม่ไปโรงเรียน” เป็นเงื่อนไขแบบใด?
เฉลย: ข้อความนี้เป็นเงื่อนไขถ้า-แล้ว (If-then condition)
โจทย์ 4: ในตรรกศาสตร์ คำว่า “และ” ใช้แทนด้วยสัญลักษณ์ใด?
เฉลย: “และ” ในตรรกศาสตร์แทนด้วยสัญลักษณ์ ∧
